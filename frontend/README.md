# Frontend

Frontend de `CatalogosYa` construido con React, Vite y TailwindCSS.

## Incluye

- React con Vite
- TailwindCSS
- React Router
- Axios con manejo de JWT
- SweetAlert2 e iziToast
- React Hook Form
- Lucide React

## Rutas base

- `/login`
- `/admin`
- `/admin/tiendas`
- `/admin/tiendas/:id`
- `/admin/categorias`
- `/admin/productos`
- `/:slug`
