# CatalogosYa

Base inicial del proyecto `CatalogosYa`, un SaaS para gestionar catalogos online.

## Alcance de esta fase

En esta etapa solo se define la estructura principal del proyecto:

- `backend/`
- `frontend/`

No se agregan frameworks, dependencias ni implementaciones adicionales para no avanzar a otra fase.

## Reglas del sistema

- Dominio unico: `catalogosYa.com`
- Cada tienda tendra una URL publica por slug
- Ejemplo de URL publica: `catalogosYa.com/tienda-rouss`
- No se usaran subdominios
- El backend debe mantenerse independiente del frontend
